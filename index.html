Entorno virtual

El entorno virtual es una forma de instalar librerias en una parte creada para su entorno especifico. Esto ayuda a que podamos exportar un proyecto con las librarias en su interior

  1. Instalar virtualenv pip install virtualenv
  2. Crear un entorno virtual virtualenv venv
  3. Activar el entorno virtual .\venv\Script\activate
  4. Desactivar entorno virtual deactivate

Instalar Kivy

Si ya tenemos listo los pasos anteriores, instalamos la libreria kivy pip install kivy

Sintaxis Básica

                        import kivy.app from App

                        class Nombre_App(App):
                            pass
                        
                        Nombre_App().run()
  • Importamos la clase App del archivo kivy/app.py from kivy.app.
  • Heredamos las variables de la clase (App) en nuestra nueva clase Nombre_App.
  • La palabra clave pass permite las clases vacias.
  • Arrancamos la clase ProyectApp() con la función run().
